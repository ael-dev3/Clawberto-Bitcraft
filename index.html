Research notes

  • Frontend studied: SvelteKit plus Leaflet plus WebSocket plus Cloudflare.
  • Live player feed comes from wss://live.bitjita.com.
  • Public player search/details come from /api/players on bitcraftmap.com.
  • Region resource GeoJSON comes from https://bcmap-api.bitjita.com/region{n}/resource/{id}.
  • That resource API is CORS-locked to bitcraftmap.com, so this hosted app uses local cached snapshots for example resources.
  • This hosted build is intentionally cropped and locked to region 12 only.
